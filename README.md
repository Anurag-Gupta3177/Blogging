Blogging Website

This is a simple blogging website where users can register, log in, view blog posts, and leave comments. The project demonstrates basic functionality for user authentication and interaction with blogs, allowing users to engage with content by commenting on posts.

Features:

1. User Authentication: Users can sign up, log in, and log out securely.
2. View Blogs: Logged-in users can view all available blog posts with titles, images, and content.
3. Commenting: Users can add comments to blog posts, and view others' comments in real-time.
4. Responsive Design: The website is designed to be user-friendly across different devices.
   
Technologies Used:

1. Node.js and Express for the backend.
2. MongoDB with Mongoose for database management.
3. EJS templating for the front-end.
4. Bootstrap for the UI.
